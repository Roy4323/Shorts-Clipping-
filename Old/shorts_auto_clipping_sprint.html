⚡ 8-Hour Sprint Plan

Shorts Auto-Clipping Hackathon Execution

🧠 Core Principle

Do NOT split work by roles. Split by independent pipeline stages and merge quickly.

Goal: Working demo > perfect system

🕐 Phase 0 (0 - 0.5 hr)

  • Lock scope: YouTube URL → 1-2 clips
  • Remove heavy components (diarization, embeddings)
  • Define final output format

🚀 Phase 1 (0.5 - 2.5 hrs)

Track A - Video Pipeline

  • yt-dlp download (720p)
  • ffmpeg clip extraction
  • Test with hardcoded timestamps

Track B - Transcript + Scoring

  • Fetch transcript
  • Basic LLM scoring
  • Return top 2 timestamps

✅ Checkpoint: Generate clips from timestamps

🚀 Phase 2 (2.5 - 5 hrs)

Track A - Reframing

  • Center crop (baseline)
  • Face tracking (optional)
  • Vertical video output

Track B - Smarter Scoring

  • Add audio energy scoring
  • Combine with semantic score
  • Improve clip quality

✅ Checkpoint: End-to-end smart pipeline working

🚀 Phase 3 (5 - 7 hrs)

Track A - Subtitles

  • Generate SRT
  • Burn subtitles using ffmpeg

Track B - AI Agent Layer

  • Add reasoning for clips
  • Return structured output

✅ Checkpoint: Clips + subtitles + reasoning

🏁 Final Phase (7 - 8 hrs)

  • Return only best clip
  • Add title, duration, reasoning
  • Test full flow

❌ Avoid These

  • Speaker diarization
  • Optical flow tracking
  • Sentence embeddings
  • Async infra (Celery)

🎤 Demo Pitch Line

We built a cost-efficient AI pipeline that uses semantic understanding and audio signals to detect high-engagement moments, avoiding expensive video AI APIs.
